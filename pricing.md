# Freemium

The first and most important thing to know is that the program is distributed under the [Freemium](https://en.wikipedia.org/wiki/Freemium) model.

In practice, this is similar to [Free-to-Play](https://en.wikipedia.org/wiki/Free-to-play), but for software: anyone can download and use the program at any time, for as long as they want, and a significant part of the functionality is available for free. So you can always download the program first and check whether it fits your needs.

If the base functionality eventually becomes limiting, you can purchase the modules you need. Important: even if you stop paying for a license at some point, your auras and scripts will not disappear.

## Subscription plans

At the moment, there are 3 main options:

| **License type** | **Free** | **Pro** | **Max** |
|---|---:|---:|---:|
| Number of auras/scripts | 10 | Unlimited | Unlimited |
| Maximum script size | 100 characters | Unlimited | Unlimited |
| Maximum number of computers | --- | 3 | 5 |
| Additional input methods | 5 | 7 | 10 |
| [Messaging](/actions/send-network-message) | 1/minute | 3/minute | Unlimited |
| [Private auras](/permission-model) | No | No | Yes |
| Pack functionality | No | No | Yes |

## Number of computers

Each license includes a certain number of computers. Any connections above that limit automatically receive a **Freeware** license.

You can always purchase additional connections separately if needed.

## Pricing

You can see the final price on the website after adding the selected items to the cart.

| **License type** | Discount | **Pro** | **Max** |
|---|---:|---:|---:|
| 1 month | --- | 5$ | 10$ |
| 3 months | 15% | 13$ | 25$ |
| 12 months | 20% | 50$ | 90$ |
| +2 PC / month | --- | 2$ | 3$ |
| +2 PC / 3 months | 20% | 5$ | 8$ |
| +2 PC / 12 months | 33% | 18$ | 30$ |
| +10 PC / month | 10% | 9$ | 13$ |
| +10 PC / 12 months | 33% | 90$ | 13$ |

Examples:

- I have **3** computers and want to use the **Pro** version: buying a **Pro** subscription is enough.
  - If billed monthly, the cost is **5$/month**
  - If billed yearly, the cost is **50$ = 4.2$/month**
- I have **5** computers and want to use **Pro**:
  - If billed monthly, the cost is **5$** (license) + **2$** (additional PCs) = **7$/month**
  - If billed yearly, the cost is **50$ + 18$ = 68$ = 5.6$/month**
