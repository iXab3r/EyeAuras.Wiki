# Text

Text Overlay is a simple yet effective overlay that displays a user-defined text inside an overlay. It shares all common options with other overlays which can be explored in more detail [here](https://wiki.eyeauras.net/en/overlays/common-options).

Text Overlay provides a set of options that allow for customization to fit different use cases. These options include:

### Text

The text to show. This option doesn't have any current limitations, enabling you to input any text that suits your needs.

### Outline Thickness

This option allows you to specify the thickness of an outline around each character. You can adjust this setting to make the text more legible against different backgrounds.

### Outline Color

This option lets you select the color of the outline around each character. This feature works in combination with Outline Thickness to improve text visibility.

### Fit to Size

If enabled, this option will automatically adjust the font size depending on the overlay's content. This ensures that the text will fit inside the overlay regardless of its length.

### FontSize

If the 'Fit to Size' option is disabled, this option will come into play. Use this setting to specify a particular font size to display the text.

### FontColor

This option allows you to specify the color of the text. This setting, in combination with Outline Color, can be used to ensure that your text is always clearly visible against the overlay's background.

### Text Anchor

The Text Anchor option dictates where to dock text inside an overlay. The following anchors are available:

- TopLeft
- Top
- TopRight
- Right
- BottomRight
- Bottom
- BottomLeft
- Left
- Center

The anchor points operate similarly to the standard [overlay anchors](/en/coordinate-system), adjusting the positioning of the text within the overlay.

Whether you need to display instructions, provide information, or show updates, the Text Overlay provides a versatile solution. By combining these options, you can customize your overlays to suit a wide range of applications and preferences.

### Examples

- **In-Game Chat Notifications**: If you have specific keywords or phrases you are monitoring in the in-game chat, a text overlay can be used to alert you when these words or phrases appear. For example, in an MMORPG, you can set up an overlay to alert you when someone is looking for a group for a particular dungeon or raid.
- **Cooldown Reminders**: If you're playing a game that has ability or item cooldowns, you can set up an overlay to remind you when the cooldown has expired. This can help you maximize your ability usage in games like MOBA's and MMO's.
- **Objective Notifications**: In strategic games where objectives change dynamically, a text overlay can be used to keep track of current objectives and display them on the screen.
- **Health or Mana Low Alerts**: In games where maintaining health or mana is crucial, you could have an overlay that alerts you when your levels drop below a certain point.
- **Key Binding Reminders**: For games with complex control schemes, you can use a text overlay to remind you of key bindings for certain actions, especially if they are ones you don't use frequently.
- **Loot Alerts**: In loot-heavy games, an overlay can alert you when a high-value or rare item drops.
- **Quest or Mission Instructions**: If a game provides complex quests or missions, a text overlay can serve as a constant reminder of your specific goals or steps needed to complete the task.
- **High Score or Record Alerts**: For competitive or speed-running games, an overlay can alert you when you're approaching or have beaten a high score or record.
- **Enemy Spawn Alerts**: In games where enemies spawn at regular intervals or specific times, an overlay can alert you to prepare for the next wave or enemy spawn.
- **Strategy Reminders**: In complex strategy games, you can use text overlays to remind yourself of specific strategies or counters to certain enemy tactics.
