# Image

Image Overlay in EyeAuras allows users to display an image of their choice within an overlay. This could be a static image or an animated GIF. You can refer to the common overlay options here: [Common Options](https://wiki.eyeauras.net/en/overlays/common-options)

### Features

- User-defined image: The user can choose any image they want to display in the overlay. This image can be either a static image (e.g. PNG, JPEG) or an animated image (GIF).
- Aspect ratio: The user can choose to maintain the original aspect ratio of the image, even when the overlay is resized.

### Gaming Applications

1. Display a small map or guide on the screen while navigating complex game levels.
2. Show an animated countdown or progress bar during timed in-game events.
3. Display an image of the game's control scheme for quick reference.
4. Show an icon or visual cue when a certain game event occurs, such as an enemy's appearance.
5. Overlay a team logo or emblem during eSports broadcasts or live streams.
6. Display a "BRB" (Be Right Back) image when stepping away from a live stream.
7. Use a custom crosshair image in FPS games for better targeting.
8. Show a combo list during fighting games for quick reference.
9. Display an animated GIF as a fun reaction to in-game events during streams.
10. Use as a visual alert when a certain condition is met in the game, such as low health or ammo.
