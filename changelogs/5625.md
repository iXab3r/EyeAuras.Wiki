# Auto-update improvements

A new option is now available in **Settings**: **Automatic update downloads**.

When enabled, new versions will be downloaded in the background as soon as they become available, so you can install updates much faster than before.

- You can still postpone the update if you're not ready to install it yet. The system will wait for your confirmation.
- Want to see what’s included in the update? Click the cloud icon to open a pop-up with detailed information.
- The pop-up behavior has also been improved to make it less intrusive. Instead of showing notifications for every minor update or informational event, you’ll now see a subtle notification next to the small cloud icon in the window header.
- Error notifications have been improved as well. Short error messages will appear next to the cloud icon, while detailed error information will be shown in the pop-up window.
- The changelog has also been rewritten from scratch. Feel free to share your thoughts on the new UX — your feedback matters.

## Bugfixes / Improvements

- **[SendInput] [Usb2Kbd]** Added a new **CharSet** option that lets you choose between **ANSI** and **Unicode**. This change improves compatibility with different OS versions. The default setting is **ANSI**, which should be sufficient for most users.
