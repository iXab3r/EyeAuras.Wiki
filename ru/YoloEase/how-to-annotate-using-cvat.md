# Аннотация изображений

Аннотация — это процесс разметки объектов интереса на изображении. По сути, вы показываете будущей модели, какие именно объекты нужно распознавать и как они выглядят. Пока для этого мы будем использовать CVAT, так как у него довольно удобный редактор.

1. Нажмите `N` или выберите **Draw new rectangle** в левой части редактора.
2. Выделите нужный объект и назначьте ему правильную метку, если используется несколько меток.
3. Повторите то же самое для остальных объектов.
4. Вот как может выглядеть итоговый результат — здесь я отметил 3 сферы разного размера: https://i.imgur.com/AFwAOaj.jpeg
5. Когда закончите с аннотацией, нажмите **Menu => Finish the job**.

[![Finish the job](https://i.imgur.com/uTC0uYw.png =x200)](https://i.imgur.com/uTC0uYw.png)

> Не забудьте сделать это, иначе YoloEase не будет считать эту задачу корректным источником аннотаций!
