# YoloEase

YoloEase — это инструмент, который автоматизирует процесс обучения Yolo 8+. Он умеет использовать возможности разметки из [CVAT](https://www.cvat.ai/) (подойдет любой сервер; по умолчанию используется [неофициальный](https://cvat.eyeauras.net/) инстанс, но вы можете сменить его в любой момент).

Wiki находится здесь: https://wiki.eyeauras.net/ru/YoloEase/getting-started

> Инструмент пока находится на ранней стадии alpha. Будем очень благодарны за любой [отзыв](https://wiki.eyeauras.net/ru/contacts)
{.is-warning}

## С чего начать

1. Установите необходимые [предварительные компоненты](/ru/YoloEase/prerequisites).
2. Скачайте последнюю версию [здесь](https://github.com/iXab3r/YoloEase/releases/).
3. Настройте проекты [CVAT](/ru/YoloEase/how-to-setup-project) и YoloEase. Они будут использоваться вместе на разных этапах процесса.
4. Перейдите к [обучению](/ru/YoloEase/how-to-use-automatic-trainer) с помощью автоматического тренера.
5. Разверните и используйте свою обученную модель.

Сначала настраиваются параметры CVAT, датасета и модели.

![Main window](https://i.imgur.com/Mm7J8nc.png =x400)

После этого можно начинать создавать задачи на разметку. При создании новой задачи можно выбрать предразметку батча и/или отобрать только те изображения, которые принесут модели наибольшую пользу.

![Task Settings](https://i.imgur.com/a9owC7c.png =x400)

Как только программа обнаружит изменения — в настройках, разметке или изображениях — она сразу же переобучит модель. Когда модель будет готова, ее можно использовать для предразметки следующего батча.

![Training process](https://i.imgur.com/gunKrAJ.png =x500)

## Как это упрощает процесс

- **Датасеты**: выберите неразмеченные изображения для обучения. YoloEase работает с ними в режиме только для чтения. Поддерживаются изображения и видео.
- **Управление изображениями**: YoloEase автоматически распределяет изображения по категориям (например, размеченные, неразмеченные, «битые», «выбросы»), чтобы ресурсы использовались эффективно и не было лишней работы. От вас требуется только периодически добавлять новые изображения.
- **Конфигурация**: задайте базовую модель, параметры обучения и другие настройки. Эта конфигурация будет использоваться, как только данные будут подготовлены.
- **Цикл разметки**: размечайте набор изображений в CVAT, а YoloEase автоматически переобучит модель, как только обнаружит появление новой разметки.
