# Свойства по умолчанию

## Настройка через интерфейс

Свойства по умолчанию появились как ответ на давнюю проблему в EyeAuras: «Как быстро и удобно изменить сразу много свойств у ВСЕХ аур?». Раньше для этого приходилось использовать разные обходные варианты — клонирование, Bindings, скрипты. Все они работали, но были недостаточно удобны, особенно для новых пользователей.

Свойства по умолчанию решают эту задачу и делают простые массовые операции действительно простыми. Например, можно сразу поменять `Target Window` у всех аур в папке.

Эта функция напрямую привязана к папкам, поэтому для её использования ауры должны находиться внутри папки. В целом это логично: EyeAuras и так предполагает, что ауры лучше держать в организованном виде.

После создания папки у неё появится отдельный раздел **“Triggers/Actions/Overlays Defaults”**:

![](/trtr5ma[1].png)

В этом разделе отображаются свойства, которые можно задать как значения по умолчанию. Технически сюда можно вынести **любое** свойство любого типа trigger/action/overlay, но для начала были добавлены 3 наиболее часто запрашиваемых варианта.

У каждого свойства есть чекбокс. Он, по сути, «включает» распространение выбранного значения из папки на все ауры внутри неё. Если все чекбоксы выключены, поведение остаётся точно таким же, как было до появления этой функции.

Если вы задали для свойства значение по умолчанию, то у всех **новых** triggers/actions/overlays это свойство сразу будет установлено в значение, указанное на уровне папки. Более того, если позже вы поменяете значение по умолчанию, дочерние ауры автоматически обновятся.

Для **уже существующих** аур ничего автоматически не меняется. Чтобы сбросить свойство trigger к значению по умолчанию, есть три способа:

1. Нажать правой кнопкой мыши на маленький чёрный квадрат справа от свойства и выбрать **"Reset to Default"**  
   (доступно только если нет активного binding)
2. Нажать левой кнопкой мыши на тот же квадрат и в редакторе bindings выбрать **“Default”**
3. На странице папки навести курсор на нужное свойство и нажать **"Apply to All"**  
   Это сбросит свойства у **ВСЕХ** дочерних аур к значению по умолчанию — по сути, то же самое, что нажать **"Reset to Default"** вручную для каждой из них
