# Что дают скрипты в BankaBot
Скрипты в `BankaBot` это не "добавить одну команду сверху". По сути это полноценный способ писать свою логику поверх бота.

С их помощью можно:
- реагировать на события в игре
- управлять персонажем и окном игры
- рисовать подсказки и метки поверх экрана
- включать и выключать части логики по ситуации
- собирать свои автоматизации, которые трудно выразить одними нодами

# Чем это отличается от обычных "бот-скриптов"
Здесь скрипты это не маленький язык с кучей ограничений, а нормальный `C#`.

На практике это значит:
- можно писать не только короткие куски логики, но и полноценные мини-программы
- можно подключать `NuGet`-пакеты
- можно держать код в более вменяемой структуре, а не в одном гигантском файле
- можно сочетать скрипты с деревьями поведения, а не выбирать что-то одно

И важный факт: сам `BankaBot` по своей природе тоже работает как скрипт внутри `EyeAuras`. Это хорошо показывает уровень возможностей местной скриптовой системы.

# Что скрипт видит и чем может пользоваться
Внутри у скрипта есть отдельная среда, через которую он получает доступ к нужным вещам:
- персонаж, цель, группа, окна игры
- ввод с клавиатуры и мыши
- `OSD`, то есть рисование поверх игры
- реакции на таймеры
- реакции на системные сообщения
- реакции на сетевые сообщения
- реакции на нажатия клавиш
- реакции на телепорт или резкую смену позиции

Это удобно, потому что не нужно каждый раз строить все с нуля. Большая часть полезных заготовок уже есть.

# Группы скриптов
Одна из полезных идей в `BankaBot` это группы скриптов.

Смысл простой: большой сценарий можно разбить на части. Например:
- бой
- бафы
- подбор лута
- уведомления
- перемещение

После этого такие группы можно включать и выключать отдельно. Это удобно и для отладки, и для повседневного использования. Не обязательно каждый раз останавливать весь скрипт целиком, если хочется временно отключить только одну часть поведения.

В интерфейсе для этого есть отдельные виджеты-кнопки, чтобы быстро щелкать группы прямо во время работы.

# Где здесь AI
Важная фишка `BankaBot` это глубокая интеграция с `AI Code Assistant`.

Если ты не хочешь писать код с нуля, это сильно упрощает жизнь:
- можно попросить помочь написать новый кусок логики
- можно попросить переделать уже существующий скрипт
- можно задавать вопросы по API, поведению бота и возможностям скриптов

Это не отменяет необходимость проверять результат головой, но порог входа становится заметно ниже.

# Когда скрипт действительно нужен
Скрипты особенно полезны, если:
- логика слишком сложная для одного дерева поведения
- нужны таймеры, события и несколько независимых частей поведения
- нужно нестандартное поведение под свой сервер или стиль игры
- хочется добавить свои оверлеи, подсказки или служебную логику

Если задача простая, начни с дерева поведения. Если уперся в потолок, тогда уже добавляй скрипт.
