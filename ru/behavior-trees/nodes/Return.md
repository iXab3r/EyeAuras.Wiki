# Return

**Return** — это узел, который позволяет немедленно завершить выполнение макроса с указанным статусом, подобно оператору `return` в C#. Доступен только внутри макросов.

## Когда использовать

Иногда макрос нужно завершить раньше, не дожидаясь выполнения всех шагов — например, если что-то пошло не так или нужное условие уже выполнено. В таких случаях используется Return:

- чтобы остановить макрос при ошибке или неудачном результате;
- чтобы прервать выполнение при выполнении определённого условия.

## Как работает

- Добавьте этот узел в нужное место макроса.
- Укажите статус, с которым должен завершиться макрос: `Success` или `Failure`.
- Как только срабатывает Return, макрос немедленно завершается, и всё, что находится после него, **не будет выполнено**.

## Пример

```csharp
IfThenElse
├── Condition: No mana
├── If yes: Return (Result = Failure)
└── If no: Continue with other actions
```

В этом примере, если маны нет, макрос завершается со статусом `Failure`, а все последующие шаги пропускаются.
