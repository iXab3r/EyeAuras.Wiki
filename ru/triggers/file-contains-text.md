# File Contains Text

**File Contains Text** отслеживает изменения в указанном файле и срабатывает по заданным условиям. Этот триггер особенно полезен для контроля конфигурационных файлов, логов и любых других документов, где изменения содержимого могут означать важные события или запуск нужных действий.

![Триггер File Contains Text](https://s3.eyeauras.net/media/2024/05/EyeAuras_7j9OpEKOveRWtpG9.png)

## Описание

Триггер **File Contains Text** наблюдает за выбранным файлом и проверяет его содержимое с помощью [**Text Match Expression**](https://wiki.eyeauras.net/e/en/text-match-expressions). Если содержимое файла соответствует условиям, заданным в выражении, триггер активируется. Если условия не выполняются, триггер деактивируется.

Если отслеживаемый файл недоступен или не существует, статус триггера переходит в состояние `'Unknown'` до тех пор, пока файл снова не станет доступен или не будет найден.

## Параметры

**File Path:** укажите файл, который нужно отслеживать. Это можно сделать вручную, введя путь к файлу, или выбрать файл через кнопку OpenFileDialog.

[**Text Match Expression**](https://wiki.eyeauras.net/e/en/text-match-expressions)**:** задаёт условия, при которых триггер должен активироваться. Выражение может быть как простым, так и сложным — в зависимости от вашей задачи.

## Примеры использования

**Изменения в конфигурационных файлах:** если приложение регулярно обновляет конфигурационный файл, триггер может уведомить вас о появлении нужных изменений.

**Мониторинг логов:** настройте Text Match Expression, чтобы получать уведомление при появлении определённой ошибки в логе или отслеживать конкретные события.

**Обновления документов:** отслеживайте изменения в документе и запускайте действия, когда в него добавляется или из него удаляется определённый текст.

## Примечание

Триггер **File Contains Text** отслеживает только один файл за раз. Чтобы триггер оставался активным, важно, чтобы этот файл не был перемещён, переименован или удалён. Если это всё же произойдёт, статус триггера переключится в `'Unknown'` до устранения проблемы.
