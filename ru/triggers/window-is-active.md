# `Window Is Active`

Триггер `**Window Is Active**` используется для определения того, находится ли конкретное окно, заданное через Window Matching Expression, на переднем плане и является ли оно активным.

![Window Is Active Trigger](https://s3.eyeauras.net/media/2024/05/EyeAuras_TntB9EP1WPMvNOEe.png)

## Обзор

У этого триггера есть три состояния:

1. **Active** — когда указанное окно находится на переднем плане и активно, состояние триггера будет `**Active**`.
2. **Inactive** — если окно свернуто или неактивно, состояние триггера переключается в `**Inactive**`.
3. **Unknown** — если указанного окна вообще нет в системе, состояние триггера переключается в `**Unknown**`. Это означает, что EyeAuras не может определить, активно окно или нет.

## Параметры

- **Target Window** — позволяет указать окно, состояние которого нужно проверять. Для этого используется Window Matching Expression. Подробнее о Window Matching Expressions читайте на нашей [_wiki page_](https://wiki.eyeauras.net/e/en/window-matching-expressions).

## Примеры использования

Этот триггер удобно использовать, чтобы сделать рабочее пространство более динамичным и адаптировать его к текущему активному окну.

- **Пользовательские хоткеи для FPS**: допустим, у вас есть любимый шутер от первого лица, например CS:GO. Вы хотите назначить горячую клавишу, которая показывает оверлей с картой текущего уровня, но только когда активно окно CS:GO. Для этого вы создаете ауру с триггером `**Window Is Active**`, используя Window Matching Expression для CS:GO. Затем привязываете хоткей к этой ауре, и он будет работать только тогда, когда активно окно CS:GO.
- **Быстрые макросы для MMORPG**: в MMORPG, например World of Warcraft, у вас может быть несколько часто используемых макросов или специальных действий. Вы можете настроить ауры с триггером `**Window Is Active**` для World of Warcraft и привязать к каждой ауре отдельный хоткей. Эти хоткеи будут запускать быстрые действия или макросы только тогда, когда World of Warcraft является активным окном.
- **Настройка автомобиля в гоночной игре**: в таких играх, как Forza Horizon, может понадобиться хоткей для быстрого открытия меню настройки или кастомизации автомобиля. Вы настраиваете ауру с триггером `**Window Is Active**` для Forza Horizon и привязываете к ней хоткей. После этого хоткей будет работать только тогда, когда активно окно Forza Horizon.
- **Переключение интерфейса для Fortnite**: допустим, вы хотите использовать определенный layout или HUD только во время игры в Fortnite. Для этого задайте `**Window Is Active**` с Window Matching Expression, который соответствует окну Fortnite. Когда Fortnite становится активным окном и выходит на передний план, EyeAuras активирует этот триггер и загрузит выбранный layout.
- **Оповещения для стрима**: если вы стримите игру и хотите запускать определенное оповещение или эффект при открытии конкретной игры, настройте `**Window Is Active**` с Window Matching Expression для этой игры. Когда игра станет активным окном, EyeAuras сможет запустить оповещение, включить нужный layout для стрима или выполнить любое другое заданное действие.
