Этот триггер позволяет настроить реакцию на получение сообщений в выбранной группе Telegram.

> Используйте действие [Send Telegram Message](/ru/actions/send-telegram-message), чтобы автоматически отправлять уведомления о _чём-либо_
{.is-info}

# Настройки триггера

Как и другие триггеры с сопоставлением текста, этот триггер поддерживает [Text Match Expressions](https://wiki.eyeauras.net/en/text-match-expressions).

Перед использованием триггера нужно создать Telegram-бота, через которого EyeAuras будет получать сообщения из группы. В целом это несложно, хотя в первый раз процесс может показаться запутанным. К счастью, в интернете есть немало инструкций.

Самая важная часть настройки — получить `Token`, который EyeAuras будет использовать для работы от имени бота и чтения сообщений из канала.

Полезные материалы по Telegram-ботам:
- Официальная документация, раздел BotFather: [Документация Telegram: Боты (tlgrm.ru)](https://tlgrm.ru/docs/bots#botfather)
- Инструкция с иллюстрациями по настройке чат-бота: [Как подключитьTelegram чат-бот | SendPulse](https://sendpulse.com/ru/knowledge-base/chatbot/create-telegram-chatbot)

# Настройка Telegram-бота и группы

Чтобы настроить уведомления через Telegram, нужно сделать 3 шага:

1. Создать Telegram-бота — он будет использоваться для отправки уведомлений. На этом шаге вы получите `Telegram Bot Token`. Можно использовать существующего бота или создать нового по инструкции ниже.
2. Создать группу или добавить своего Telegram-бота в уже существующую группу — именно туда будут отправляться сообщения. Чтобы получить `Telegram Chat Id` группы, можно добавить ещё одного бота: `Show Json Bot`.
3. Настроить Send Telegram Message, указав `Telegram Bot Token` и `Telegram Chat Id`.

### Как получить Telegram Bot Token

1. Откройте Telegram-клиент — настольный или мобильный, разницы нет.
2. Найдите бота `BotFather`.
3. Отправьте команду `/newbot`.
4. Придумайте и введите имя для нового бота. Оно не обязано быть уникальным, например: `MyAwesomeNewTelegramBot`.
5. Придумайте и введите username для нового бота. Он должен быть уникальным и обязательно заканчиваться на `bot`, например `MyAwesomeNewTelegramBot` (скорее всего это имя уже занято, поэтому придётся выбрать другое).
6. Готово. Если всё сделано правильно, `BotFather` пришлёт вам `Bot Token`.

### Как получить Chat Id

1. Создайте новую группу в Telegram или откройте существующую.
2. Нажмите кнопку **Add Member**.
3. Найдите и добавьте `YOUR_BOT_NAME` (username бота, которого вы создали ранее, например `MyAwesomeNewTelegramBot`).
4. Найдите и добавьте `Show Json Bot` (это важно: имя должно быть именно таким, потому что есть много похожих ботов).
5. (необязательно) Нажмите Create — это нужно только если вы создаёте новую группу с нуля.
6. **Show Json Bot** отправит JSON, в котором будет указан ChatId.
