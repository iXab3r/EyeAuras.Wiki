# Выражения для поиска окон

`Window Matching Expression Control` — это концепция и элемент интерфейса в EyeAuras. С его помощью можно задать выражение для выбора одного или нескольких окон по разным критериям.

Такие выражения поддерживаются во всех actions и triggers в EyeAuras. Они используются, когда нужно указать:

- куда отправлять ввод
- какое окно должно быть активным
- с какого окна делать скриншот или запись
- и в целом с каким окном должна работать функция

![](/eyeauras_zjebk8jadf.png)

## Из чего состоит этот элемент

У этого контрола есть три части:

**Выпадающий список выбора окна**  
Нажатие на маленькую иконку открывает список всех активных окон. Если выбрать окно из списка, EyeAuras автоматически создаст выражение поиска — по заголовку окна или по его handle.

**Поле ввода выражения**  
Текстовое поле, в которое можно вручную ввести выражение для поиска окна. EyeAuras разбирает это выражение и определяет, какое окно или окна нужно выбрать.

**Область выбранного окна**  
Здесь показывается окно или список окон, найденных по заданному выражению. Если навести курсор, появится tooltip с подробной информацией об окне — это помогает точнее настроить выражение.

## Как создавать выражения

Выражение можно ввести вручную в поле ввода. EyeAuras интерпретирует его как правило поиска окна.

### Обычный текст

Если ввести просто `test`, будут найдены все окна, в заголовке которых есть `test`.

Используется частичное совпадение без учета регистра.

### Точное совпадение текста

Если заключить текст в одинарные (`' '`) или двойные (`" "`) кавычки, будет использоваться точное совпадение с учетом регистра.

Например, `'test'` найдет только окно с заголовком `test`.

### Регулярное выражение

Если текст записан в формате `/expression/`, EyeAuras воспринимает его как регулярное выражение и ищет окна по шаблону заголовка.

### По handle окна

Можно указать handle окна напрямую.

Поддерживаются:

- шестнадцатеричный формат, например `0xA`
- формат с префиксом `&H`, например `&HA`

В этом случае EyeAuras ищет окно по его handle.

Важно: handle может меняться при каждом перезапуске окна или приложения.

### По индексу

Если добавить к выражению `#number`, можно выбрать окно по его индексу.

Например:

- `test #2` — выберет второе окно, в заголовке которого есть `test`
- `#2` — выберет второе окно из выпадающего списка

После этого система покажет найденное окно или окна в области выбранного окна.

Этот механизм упрощает настройку EyeAuras и помогает точнее указывать, с каким окном нужно работать.

## Примеры

- Выражение: `"Explorer"`  
  Результат: находит окно с точным заголовком `Explorer`.

- Выражение: `"/fire/"`  
  Результат: находит все окна, в заголовке которых есть `fire`, без учета регистра.

- Выражение: `"Explorer" #2`  
  Результат: находит второе окно с точным заголовком `Explorer`.

- Выражение: `"Mail" #1`  
  Результат: находит первое окно с заголовком `Mail`.

- Выражение: `0x5A`  
  Результат: находит окно с handle в шестнадцатеричном формате `5A`.

- Выражение: `"paint"`  
  Результат: находит все окна, в заголовке которых есть `paint`.

- Выражение: `"/^Notepad/"`  
  Результат: находит все окна, у которых заголовок начинается с `Notepad`.

- Выражение: `&H2B`  
  Результат: находит окно с handle в шестнадцатеричном формате `2B`.

- Выражение: `"/chrome$/"`  
  Результат: находит все окна, у которых заголовок заканчивается на `chrome`.

- Выражение: `"Calculator" #3`  
  Результат: находит третье окно с точным заголовком `Calculator`.

- Выражение: `"Excel" #1`  
  Результат: находит первое окно с заголовком `Excel`.

- Выражение: `"/Microsoft/"`  
  Результат: находит все окна, в заголовке которых есть `Microsoft`, без учета регистра.

- Выражение: `0xA1F`  
  Результат: находит окно с handle в шестнадцатеричном формате `A1F`.

- Выражение: `"/^Google Chrome/"`  
  Результат: находит все окна, у которых заголовок начинается с `Google Chrome`.

- Выражение: `#5`  
  Результат: находит пятое окно в выпадающем списке.

- Выражение: `"Firefox #2"`  
  Результат: находит второе окно с точным заголовком `Firefox`.

- Выражение: `&H7C`  
  Результат: находит окно с handle в шестнадцатеричном формате `7C`.

- Выражение: `#1`  
  Результат: находит первое окно в выпадающем списке.

- Выражение: `"/.*/"`  
  Результат: находит все окна, так как это регулярное выражение подходит под любые символы.
