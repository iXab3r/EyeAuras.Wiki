# Win Activate

`Win Activate Action` — это действие, которое переводит нужное окно на передний план по заданному `Window Match Expression`.

Оно полезно и для автоматизации рабочих сценариев, и для игровых задач, когда нужно быстро переключать фокус на конкретное окно.

## Параметры

- [Window Match Expression](/en/window-matching-expressions): условие, по которому EyeAuras определяет, какое именно окно нужно активировать. Выражение может совпадать с заголовком окна, именем класса или даже процессом.

## Где это может пригодиться

- Если у вас одновременно запущено несколько окон одной игры, `Win Activate Action` позволяет автоматически переключать фокус между ними по нужным триггерам.
- При работе с несколькими мониторами можно автоматически выводить на передний план нужное приложение, например почту или мессенджер, когда приходит новое сообщение.
- Для стримеров и создателей контента можно настроить автоматическое открытие нужных окон или приложений в определённые моменты трансляции.
- Если вы работаете сразу с несколькими программами, можно сделать так, чтобы следующее нужное приложение автоматически становилось активным в зависимости от текущей задачи.
- Если открыто несколько окон браузера, можно автоматически переключаться на то, где открыт нужный сайт, по определённым условиям.
- Для разработчиков — автоматически переключаться в IDE после успешной компиляции.
- Для игроков — быстро открывать прохождение или гайд, если вы застряли на определённом моменте.
- При системной ошибке или предупреждении можно автоматически выводить на передний план нужные инструменты диагностики или панели мониторинга.
- Если завершилась долгая задача или скачивание, можно сразу показать окно с результатом или загруженным файлом.

`Win Activate Action` помогает сделать работу за компьютером заметно удобнее и открывает много вариантов для автоматизации под ваши собственные сценарии. Иногда достаточно всего нескольких настроек, чтобы упростить повседневные действия.
