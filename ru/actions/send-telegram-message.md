Действие **Send Telegram Message** в EyeAuras позволяет подключиться к **Telegram** и отправлять сообщения через Telegram Bot API.

> Используйте [триггер Telegram Message](/ru/triggers/telegram-message), чтобы настроить реакции на сообщения, отправленные этим действием или другими пользователями в группе.
{.is-info}

Обратите внимание: сейчас отправка сообщений от имени реального пользователя не поддерживается.

# Настройка Telegram-бота и группы

Чтобы настроить уведомления в Telegram, нужно сделать 3 шага:

1. Создать Telegram-бота — он будет использоваться для отправки уведомлений. На этом шаге вы получите `Telegram Bot Token`. Можно использовать уже существующего бота или создать нового по инструкции ниже.
2. Создать группу или добавить Telegram-бота в уже существующую группу — именно туда будут отправляться сообщения. Чтобы получить `Telegram Chat Id` группы, можно добавить ещё одного бота с именем `Show Json Bot`.
3. Настроить действие Send Telegram Message, указав `Telegram Bot Token` и `Telegram Chat Id`.

### Как получить Telegram Bot Token

1. Откройте Telegram-клиент — подойдёт и десктопная, и мобильная версия.
2. Найдите бота `BotFather`.
3. Введите команду `/newbot`.
4. Укажите имя для нового бота. Оно не обязано быть уникальным, например: `MyAwesomeNewTelegramBot`.
5. Укажите username для нового бота. Он должен быть уникальным и обязательно заканчиваться на `bot`, например `MyAwesomeNewTelegramBot` (скорее всего это имя уже занято, поэтому придётся выбрать другое).
6. Готово. Если всё сделано правильно, `BotFather` пришлёт вам `Bot Token`.

### Как получить Chat Id

1. Создайте новую группу в Telegram или откройте существующую.
2. Нажмите **Add Member**.
3. Найдите и добавьте `YOUR_BOT_NAME` (username бота, которого вы создали ранее, например `MyAwesomeNewTelegramBot`).
4. Найдите и добавьте `Show Json Bot` (важно: имя должно быть именно таким, потому что существует много похожих ботов).
5. *(необязательно)* Нажмите Create — это нужно только при создании новой группы.
6. **Show Json Bot** отправит JSON, в котором будет содержаться ChatId.

# Параметры действия

**Telegram Bot Token:** ID созданного вами Telegram-бота. Подробнее о настройке можно прочитать здесь: [_Telegram Message Trigger_](/ru/triggers/telegram-message).

**Telegram Chat ID:** ID чата, в который нужно отправлять сообщение.

**SOCKS5 Proxy:** При необходимости можно использовать прокси для отправки сообщений. Формат прокси: `username:password@host:port`.

**Message Format:** Можно выбрать один из форматов сообщения: Text, HTML, Markdown или MarkdownV2. Кратко о каждом:

- **Text**: обычный текст без форматирования.
- **HTML**: форматирование с помощью HTML-тегов — например, жирный текст, курсив, ссылки и т. д.
- **Markdown / MarkdownV2**: форматирование с помощью синтаксиса markdown. Подробнее об особенностях markdown в Telegram смотрите в документации [_Telegram API_](https://core.telegram.org/bots/api#markdown-style).

**Text:** Текст сообщения, которое вы хотите отправить.

Сейчас ограничений на размер сообщения нет, но слишком длинные сообщения могут быть неудобны для чтения или обрезаться самим Telegram на стороне получателя.

### Примеры для игровых сценариев

**Уведомления о статусе игры:** С помощью действия Send Telegram Message можно настроить EyeAuras так, чтобы он отправлял уведомления о состоянии игры в Telegram-группу. Например, когда уровень пройден или игра сохранена, можно автоматически отправить сообщение друзьям или сокомандникам.

**Коммуникация в команде:** В командных играх EyeAuras может отправлять сообщения в Telegram-чат команды при наступлении определённых игровых событий — например, когда получен важный предмет или достигнута ключевая точка. Это помогает координироваться в реальном времени.

**Оповещения о стриме:** Стримеры могут использовать это действие для отправки сообщений в свой Telegram-канал при начале или завершении трансляции. Это помогает держать подписчиков в курсе расписания.

**Публикация игровой статистики:** EyeAuras можно настроить так, чтобы он автоматически отправлял игровую статистику — например, рекорды или полученные достижения — в Telegram-группу. Это удобно для отслеживания прогресса и соревнования с друзьями.
