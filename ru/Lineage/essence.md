# Информация по Lineage 2 Essence

Ниже — короткая инструкция по быстрому старту.

## 1. Скачайте EyeAuras

Сначала скачайте приложение EyeAuras с нашего сайта: [EyeAuras Download](https://eu.eyeauras.net/download)

## 2. Зарегистрируйтесь и купите лицензию

Создайте аккаунт и при необходимости приобретите лицензию, чтобы открыть все возможности. Начать можно здесь: [EyeAuras Login](https://eu.eyeauras.net/login)

## 3. Выберите метод ввода

В зависимости от уровня античита на сервере выберите подходящий способ ввода:

- Для серверов с **low anticheat** используйте Windows Input или Windows Message API для отправки клавиш в фоне.
- Для серверов с **Active Anticheat** или другими сильными античитами скачайте этот бесплатный драйвер для совместимости: [Download Driver](https://s3.eyeauras.net/media/2024/10/HVDK%20Standard%202.1.exe)

После установки перейдите в `C:\Program Files (x86)\ControlMyJoystick 5.5.78.50\Driver\Joystick` и запустите batch-файл удаления от имени администратора.

## 4. Решение проблем

Если у вас возникает баг, при котором камера начинает бесконечно крутиться, откройте настройки игры (L2 settings) и отключите `Enable Controller` в разделе Settings -> Configuration.

## 5. Скрипт

- Простой скрипт для Essence можно взять здесь: [Essence auto flags](https://eu.eyeauras.net/share/S20240325120506y5CPVmuwLvC3)
- Просто скопируйте ссылку и импортируйте её в EyeAuras.
- Если шаг 3 вам непонятен, можно установить TetherScript в один клик.
