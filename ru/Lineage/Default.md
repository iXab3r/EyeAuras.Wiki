## Базовая информация по Lineage 2

Чтобы быстро начать работу, выполните несколько простых шагов.

### 1. Скачайте EyeAuras

Сначала скачайте приложение EyeAuras с нашего сайта: [EyeAuras Download](https://eu.eyeauras.net/download)

### 2. Зарегистрируйтесь и приобретите лицензию

- Создайте аккаунт и при необходимости купите лицензию, чтобы открыть все возможности. Начать можно здесь: [EyeAuras Login](https://eu.eyeauras.net/login)

### 3. Выберите способ ввода

В зависимости от уровня античита на сервере выберите подходящий метод ввода:

- Для серверов со **слабым античитом**: используйте Windows Input или Windows Message API для отправки клавиш в фоне.
- Для серверов с **Active Anticheat** или другими сильными античитами: скачайте этот бесплатный драйвер для корректной совместимости. [Download Driver](https://s3.eyeauras.net/media/2024/10/HVDK%20Standard%202.1.exe)

После установки перейдите в `C:\Program Files (x86)\ControlMyJoystick 5.5.78.50\Driver\Joystick` и запустите batch-файл удаления от имени администратора.

### 4. Решение возможных проблем

Если возникает баг, при котором камера начинает бесконечно крутиться, откройте настройки игры (L2 settings) и отключите пункт "Enable Controller" в разделе Settings -> Configuration.
