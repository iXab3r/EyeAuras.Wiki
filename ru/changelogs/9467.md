# 1.9.9467 [RU](/ru/changelogs/9467)/[EN](/en/changelogs/9467)
## C# Scripting - Script Properties / NuGet
У `C#`-скриптов появилась кнопка `Properties`, которая открывает отдельное окно свойств скрипта. Внутри собраны основные настройки и зависимости: `Overview`, управление `NuGet`-пакетами, `References` для внутренних и внешних сборок, а также `Embedded Resources` для файлов, которые должны распространяться вместе со скриптом.
