# Новая функция — [История версий](/ru/features/config-rollback)

Добавлена функция [История версий](/ru/features/config-rollback), которая позволяет просматривать конфигурацию в виде снимков, сохранённых с течением времени, и управлять ими. Теперь пользователи могут выполнить Soft Reset, чтобы восстановить состояние файлов без изменения истории, или Hard Reset, чтобы полностью откатиться к конкретному коммиту, удалив все изменения, сделанные после него.

p.s. Русская версия документации доступна [здесь](https://wiki.eyeauras.net/ru/features/config-rollback)
