# Улучшения на стороне сервера

Поработали над стабильностью — это должно решить проблемы, из-за которых сервер падал под пиковой нагрузкой. Теперь вместо падений он должен просто замедляться :D

Спасибо всем, кто стал частью этой приятной проблемы — добро пожаловать на борт :)

# Портативная версия

На главную веб-страницу добавлена новая кнопка, которая позволяет скачать архив с самой свежей версией. Эта версия будет полностью портативной и будет использовать отдельную конфигурацию, не связанную с основной установкой приложения. Механизм уже несколько месяцев обкатывался в Packs, так что всё должно работать без серьёзных проблем.

p.s. Обратите внимание: портативная версия пока недоступна для Stable. Она появится сразу после того, как я переведу alpha в stable в ближайшие несколько дней.

## Исправления и улучшения

- [Scripting] В `ISendInputUnstableScriptingApi` добавлен `TargetWindow` — это позволяет использовать Windows Message API и отправлять ввод в фоновые окна.
- [UI] Улучшена обработка ошибок в диалогах Import/Export — теперь сообщения об ошибках должны быть понятнее и отображаться в более аккуратном виде.
- [UI] Добавлен бейдж, который показывает, сколько аур находится в состоянии Archived. Это богом забытая функция, которая позволяет архивировать (то есть выгружать и скрывать) ауры, которые вы сейчас не используете — это заметно ускоряет загрузку приложения.
