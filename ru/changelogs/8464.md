## Исправления и улучшения

- [Scripting] API чтения памяти теперь по умолчанию не чувствительны к регистру
- [Scripting] В API чтения памяти добавлен `ReadManaged<T>` — он работает медленнее, чем `Read<T>`, но учитывает `MarshalAs` и managed-структуры
- [Scripting] API чтения памяти теперь поддерживают цели как x86, так и x64. При этом помните, что ваш собственный процесс в любом случае x64.
- [UI] Исправлена проблема с выбором элементов в новом редакторе папок
- [Crash] Исправлен неприятный вылет в новом редакторе папок
