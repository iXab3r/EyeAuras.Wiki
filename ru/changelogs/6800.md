# Сообщения в приложении

В EyeAuras появилась система уведомлений: теперь у каждого пользователя есть свой почтовый ящик, который можно открыть в любой момент прямо в приложении.

Сейчас основное применение этой функции — обратная связь по вашим репортам. Например, вы будете видеть, когда ваш баг-репорт был закрыт, читать мой комментарий и другие связанные уведомления.
