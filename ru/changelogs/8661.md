# Макросы — узлы Return/Break

Добавлены два новых узла, которые помогут делать макросы лучше и умнее.

## Return

Узел Return позволяет выйти из макроса в любой момент — даже прямо посреди последовательности операций.  
Например, если персонаж умер, продолжать выполнение основного цикла уже не имеет особого смысла.  

Это аналог операции `return` в C#.

## Break

Break — более нишевый узел. Он позволяет выйти из текущей **области видимости** (но не из всего макроса!).  
Например, если вы находитесь внутри цикла `Repeat`, при необходимости можно сразу выйти из него.  

Это аналог операции `break` в C#.

### Behavior Tree / Macro

Добавлено много статей об узлах Behavior Tree / Macro.  
Обратите внимание: **большинство** из этих статей доступны и на русском, и на английском.

- [IsActive](https://wiki.eyeauras.net/en/behavior-trees/nodes/IsActive) — проверяет, активен ли сейчас родительский Tree/Macro
- [Interrupter](https://wiki.eyeauras.net/en/behavior-trees/nodes/Interrupter) — узел для продвинутых сценариев, который позволяет прервать выполнение Behavior Tree при выполнении условия
- [Return](https://wiki.eyeauras.net/en/behavior-trees/nodes/Return) — позволяет остановить макрос
- [Break](https://wiki.eyeauras.net/en/behavior-trees/nodes/Break) — позволяет выйти из цикла или блока кода
- [MouseMove Abs](https://wiki.eyeauras.net/en/behavior-trees/nodes/MouseMoveAbs) — перемещает курсор в нужное место на экране (или к _чему-то_)
- [MouseMove Rel](https://wiki.eyeauras.net/en/behavior-trees/nodes/MouseMoveRel) — перемещает курсор относительно его текущей позиции
- [CheckKeyState](https://wiki.eyeauras.net/en/behavior-trees/nodes/CheckKeyState) — проверяет, удерживается ли сейчас какая-либо клавиша
- [KeyPress](https://wiki.eyeauras.net/en/behavior-trees/nodes/KeyPress) — эмулирует нажатия клавиш
- [Send Text](https://wiki.eyeauras.net/en/behavior-trees/nodes/SendText) — вводит текст либо через вставку из буфера обмена, либо посимвольно
