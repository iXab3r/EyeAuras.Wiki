# MouseMove — добавлено случайное смещение

Добавлен простой способ случайным образом смещать координаты X/Y.  
В сочетании с Input Smoothing это должно сделать ввод более естественным.

![Randomize](https://s3.eyeauras.net/media/2024/12/NVIDIA_Overlay_9GBb5rNrf4o4FqUp.png)

## Исправления и улучшения

- [UI] Добавлены два новых действия — `Execute Tree` и `Execute Macro`, которые можно вызывать из Auras. Это своего рода «склейка», позволяющая связать два подхода к автоматизации, существующие в EA
- [Scripting] Изменён способ загрузки сборок — исправляет проблему с транзитивными зависимостями
- [Scripting] Открыт доступ к IPerformanceMetricsAddon — его можно использовать для отслеживания производительности приложения через скрипты
- [Scripting] Изменён порядок компиляции скриптов — исправляет проблему, из-за которой Razor иногда компилировался некорректно
