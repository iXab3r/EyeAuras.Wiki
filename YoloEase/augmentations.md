# Augmentations

The detailed page has moved to the features section: [Augmentations](/YoloEase/features/augmentations).

In short, augmentations create additional variations of already annotated images. They are applied while building the dataset before training. Use them after your first training runs, once you understand which real-world variations the model is still missing.
